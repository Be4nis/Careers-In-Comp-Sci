A Full-Stack web developer creates/codes the entire tech stack required to run a website. They need to know front end languages like HTML, CSS and JavaScript to make the website look nice, they have to know have an API like PHP that is able to transfer data into a database as well as a back-end languages like SQL or Python to store data or to actually run the server. This career is all about coding, a web designer can give you a front-end design that you have to code. In order to transfer user data to a database you have to write hundreds of lines of code and even running the server on the internet takes a lot of code.
